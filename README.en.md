Instructions for Use: 

1. Right-click on the desktop - Display settings - The scaling must be 100%.
2. The game resolution must be in a 16:9 aspect ratio. If your monitor is not, please adjust the resolution in the game to 16:9.
3. The game must be set to borderless mode; otherwise, the window cannot be pinned to the top.
4. It must be run as an administrator by right-clicking. 

Operation process:
1. Click on "Official Lineup"
2. Select a lineup. Use the mouse to control or press the up and down keys to switch lineups.
3. Click on "Squad Planning". It will automatically copy the lineup data to the official squad planning reminder.
If it fails, you can manually paste. At this time, the lineup text is already in your clipboard. 

Hotkey Instructions:
1. Press ~ (tilde) to turn off auto card drawing.
2. Press the Home key to hide/show the UI.
3. Press the Up and Down arrow keys to switch lineups.
